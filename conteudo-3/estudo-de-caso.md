# Estudos de caso (para discussão)

Use os casos abaixo para discutir em sala. A ideia é treinar o raciocínio: **qual plataforma atende melhor o objetivo e por quê?**

## Estudo de caso 1: laboratório educacional barato

Uma escola quer montar um laboratório barato para ensinar programação básica e eletrônica.

Perguntas:

- PC ou Raspberry Pi?
- Quais vantagens?
- Quais limitações?

**Resposta comentada:**

Em geral, **Raspberry Pi** tende a ser uma ótima escolha quando o foco é educação e eletrônica, porque tem baixo consumo e permite trabalhar com sensores/LEDs via GPIO. Limitações comuns: desempenho menor para tarefas pesadas e algumas limitações de software dependendo do que a escola pretende usar.

## Estudo de caso 2: ambiente pesado de desenvolvimento

Um desenvolvedor precisa trabalhar com Docker, banco de dados, IDE pesada e múltiplos serviços locais.

Perguntas:

- PC ou Raspberry Pi?
- Por quê?

**Resposta comentada:**

Em geral, **PC** é a melhor escolha porque esse cenário costuma exigir mais CPU, RAM e armazenamento rápido. O objetivo é produtividade e multitarefa intensa.

## Estudo de caso 3: monitoramento e automação com sensores

Uma empresa quer criar um sistema para monitorar temperatura e acionar um ventilador automaticamente.

Perguntas:

- PC ou Raspberry Pi?
- Quais componentes poderiam ser conectados?

**Resposta comentada:**

Em geral, **Raspberry Pi** é uma escolha excelente: baixo consumo para ficar ligado 24/7 e GPIO para integração com sensores e atuadores. Componentes comuns: sensor de temperatura, módulo relé para acionar o ventilador (com os cuidados elétricos adequados), e eventualmente display/LEDs para status.

## Estudo de caso 4: jogos e edição de vídeo

Um usuário quer montar um computador para jogos e edição de vídeo.

Perguntas:

- PC ou Raspberry Pi?
- Quais componentes são importantes?

**Resposta comentada:**

Em geral, **PC** é a escolha natural por exigir GPU e desempenho. Componentes importantes: CPU adequada, **GPU dedicada**, boa quantidade de RAM e armazenamento (SSD) para carregar projetos e mídia com agilidade.

## Estudo de caso 5: servidor web simples em casa

Um aluno quer deixar um servidor web simples rodando em casa para aprender Linux e redes.

Perguntas:

- PC ou Raspberry Pi?
- Quais cuidados devem ser considerados?

**Resposta comentada:**

Em geral, **Raspberry Pi** é uma ótima escolha pelo baixo consumo e custo. Cuidados: boa alimentação elétrica, armazenamento confiável (microSD de qualidade ou SSD via USB), manter o sistema atualizado e cuidar de segurança (senhas, portas, acesso remoto).
