# PC ou Raspberry Pi: como escolher?

Até aqui, você viu que não existe “melhor absoluto”.

A melhor escolha depende do **objetivo**.

## Ideia central

- O **PC** tende a ser melhor quando o foco é **desempenho**, **expansão** e **compatibilidade** com softwares tradicionais.
- O **Raspberry Pi** tende a ser melhor quando o foco é **baixo custo**, **baixo consumo**, **educação**, **prototipagem** e integração com hardware externo (**GPIO**).

## Perguntas para guiar a escolha

Antes de escolher, faça perguntas como:

- Preciso de alto desempenho?
- Preciso rodar softwares pesados?
- Preciso de baixo consumo de energia?
- Preciso conectar sensores e atuadores?
- Preciso de portabilidade?
- Preciso gastar pouco?
- Preciso de compatibilidade com softwares tradicionais?
- Preciso deixar o equipamento ligado 24 horas por dia?

## Tabela de decisão (bem prática)

| Se você precisa de... | Considere usar... |
|---|---|
| Alto desempenho gráfico | PC |
| Baixo consumo | Raspberry Pi |
| GPIO para sensores | Raspberry Pi |
| Compatibilidade com softwares profissionais | PC |
| Prototipagem rápida | Raspberry Pi |
| Jogos pesados | PC |
| Servidor simples e barato | Raspberry Pi |
| Virtualização pesada | PC |

## Lembrete importante

Existem:

- diferentes configurações de PC (de básico a workstation);
- diferentes modelos de Raspberry Pi.

Então a comparação aqui é **introdutória e geral**: o objetivo é entender o “porquê” das escolhas.
