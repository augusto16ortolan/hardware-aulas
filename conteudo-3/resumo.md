# Resumo

## Recapitulando os pontos principais

- **PC e Raspberry Pi são computadores**, mas foram projetados para objetivos diferentes.
- O **PC** costuma priorizar **desempenho**, **expansão** e **compatibilidade**.
- O **Raspberry Pi** costuma priorizar **baixo consumo**, **baixo custo** e **prototipagem**.
- Raspberry Pi é muito usado em **educação**, **IoT** e **automação**.
- PC é mais adequado para tarefas pesadas, **jogos**, **edição**, **virtualização** e uso profissional.
- A melhor escolha depende da **aplicação**.

## Tabela resumida

| Critério | PC | Raspberry Pi |
|---|---|---|
| Arquitetura do processador | Geralmente x86/x86-64 | Geralmente ARM |
| Desempenho | Geralmente maior | Menor, mas suficiente para muitos projetos |
| Memória | Mais RAM e mais opções de upgrade | Mais limitada (varia por modelo) |
| Armazenamento | SSD/HD e expansão interna | microSD e expansão via USB |
| Expansibilidade | Alta (principalmente desktop) | Limitada |
| GPIO | Não é foco | Sim, é diferencial |
| Consumo de energia | Geralmente maior | Geralmente menor |
| Desempenho gráfico | Pode ser muito alto (GPU dedicada) | Mais modesto (integrado) |
| Custo | Pode variar muito (pode ser alto) | Geralmente mais acessível |
| Aplicações principais | Profissional, jogos, edição, VMs | Educação, IoT, automação, protótipos |

## Frases‑chave para memorização

- PC e Raspberry Pi são computadores, mas possuem propostas diferentes.
- O PC prioriza desempenho, expansão e compatibilidade.
- O Raspberry Pi prioriza baixo consumo, baixo custo e prototipagem.
- Raspberry Pi é excelente para educação, IoT e automação.
- PC é mais adequado para tarefas pesadas, jogos, edição, virtualização e uso profissional.
- A melhor escolha depende da aplicação.

## Conclusão didática

Quando você entende o propósito de cada plataforma, fica mais fácil tomar decisões.

Em Hardware, isso é essencial: **tecnologia não existe no vácuo** — ela existe para resolver problemas reais com restrições reais (custo, energia, espaço, desempenho e compatibilidade).
