# Desempenho gráfico

Quando falamos em “gráficos”, estamos falando principalmente do trabalho feito pela **GPU** (unidade de processamento gráfico).

## GPU integrada vs GPU dedicada

- **GPU integrada:** vem junto com o processador (ou muito próxima dele). É suficiente para tarefas comuns (interface do sistema, vídeo, gráficos leves), mas tem limites em jogos pesados e aplicações profissionais.
- **GPU dedicada:** é uma placa específica para gráficos, com mais poder de processamento e memória própria (VRAM). É muito usada em jogos, edição de vídeo, modelagem 3D e aplicações que se beneficiam de processamento paralelo.

## PCs: geralmente melhores para gráficos pesados

PCs podem ter:

- GPUs dedicadas (por exemplo, de fabricantes como NVIDIA e AMD) **ou** GPUs integradas mais robustas (dependendo do modelo).
- mais recursos para processamento gráfico e, em desktops, possibilidade de upgrade.

Por isso, PCs são mais adequados para:

- jogos pesados;
- edição de vídeo;
- modelagem 3D e renderização;
- aplicações gráficas profissionais.

## Raspberry Pi: gráficos mais modestos (mas úteis)

O Raspberry Pi normalmente usa um chip gráfico **integrado**.

Ele pode:

- reproduzir vídeos;
- rodar interfaces gráficas simples;
- servir como media center;
- rodar jogos leves e projetos visuais simples.

Mas não é uma boa escolha para:

- jogos pesados;
- renderização 3D complexa;
- edição de vídeo pesada.

## Comparação (tabela)

| Aspecto gráfico | PC | Raspberry Pi |
|---|---|---|
| Tipo de GPU | Integrada e/ou dedicada | Integrada |
| Potência gráfica | Geralmente maior | Mais limitada |
| Uso típico | Jogos, edição, 3D, aplicações profissionais | Vídeo, interface leve, projetos simples, media center |
| Upgrade | Possível em muitos desktops | Não é o foco |

## Observação importante

Não faz sentido comparar um Raspberry Pi com um PC gamer ou workstation como se fossem o mesmo produto.

A comparação correta é: **qual plataforma atende melhor o objetivo do projeto?**
