# Visão geral: PC e Raspberry Pi

Antes de comparar peças e números, vamos entender o que cada plataforma é.

## O que é um PC?

**PC** (computador pessoal) é um computador de **propósito geral**: ele foi pensado para executar muitos tipos de tarefas diferentes, como:

- navegação na internet;
- estudos e trabalhos acadêmicos;
- desenvolvimento de software;
- jogos;
- edição de imagens e vídeos;
- aplicações profissionais (engenharia, design, escritório).

Um PC costuma ter foco em **desempenho**, **compatibilidade** e **possibilidade de upgrade/expansão**.

## O que é um Raspberry Pi?

O **Raspberry Pi** é uma **placa de computação compacta** (um computador em uma placa pequena) muito usada para:

- educação (programação e fundamentos);
- prototipagem de projetos;
- automação e IoT (Internet das Coisas);
- pequenos servidores (simples, de baixo custo);
- integração com sensores e componentes eletrônicos.

Ele costuma ter foco em **baixo custo**, **baixo consumo de energia** e **facilidade para projetos práticos**.

## Por que ambos são computadores?

Em termos simples, um computador precisa de:

- um **processador** (CPU) para executar instruções;
- **memória RAM** para manter dados temporários enquanto roda programas;
- **armazenamento** para guardar sistema e arquivos;
- **entradas e saídas** (I/O) para se comunicar com o mundo (teclado, rede, tela, etc.).

Tanto o PC quanto o Raspberry Pi têm esses elementos — o que muda é **como** e **por quê** foram projetados.

## Propósito geral vs. placa compacta (ideia central)

- Um **PC tradicional** costuma ser usado para tarefas variadas e mais pesadas.
- Um **Raspberry Pi** costuma ser usado para projetos menores, experimentação, automação, servidores simples e aprendizado.

Importante:

- O Raspberry Pi **não substitui um PC em todos os cenários**.
- O PC **não é sempre a melhor escolha** para projetos simples ou embarcados.

## Comparação inicial

| Característica | PC | Raspberry Pi |
|---|---|---|
| Tamanho | Maior (gabinete/notebook) | Pequeno (placa) |
| Consumo de energia | Geralmente maior | Geralmente menor |
| Custo | Varia bastante; pode ser alto | Geralmente mais acessível |
| Desempenho | Geralmente maior | Menor, mas suficiente para muitos projetos |
| Expansibilidade | Alta (troca/adição de peças) | Limitada; expansão via acessórios e módulos |
| Uso educacional | Muito comum | Muito comum (especialmente com eletrônica/IoT) |
| Uso profissional | Muito comum | Em nichos específicos |
| Facilidade para prototipagem | Média (depende do objetivo) | Alta (GPIO e projetos físicos) |
